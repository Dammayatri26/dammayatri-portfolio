## Overview

Geocoded patient address data from all branch clinics into spatial format using the Google Maps API to map and visualize patient distribution patterns, providing insight for other departments' marketing and expansion planning.

## Methods & Tools

**Data Sources**

- Patient address records from all branch clinics

**Processing Steps**

1. Collected patient address data across all branch clinics.
2. Geocoded addresses into spatial coordinates using the Google Maps API.
3. Visualized the geocoded data to map patient distribution and identify spatial patterns.
4. Cleaned the outlier data
5. Shared distribution insights with other departments to support marketing and expansion decisions.

**Tools Used**

| Tool | Purpose |
|------|---------|
| Google Maps API | Geocoding patient addresses into spatial coordinates |
| ArcGIS Pro and Felt | Data visualization and mapping |

---
## Key Findings

- Revealed spatial clustering of patients around for all Satu Dental Branches
- Identified underserved areas with limited patient coverage relative to clinic locations.
- Findings informed marketing and expansion planning discussions for other departments.

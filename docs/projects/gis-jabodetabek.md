## Overview

Used spatial analysis and a hexagonal grid framework to evaluate population density, competitor locations, and demographic demand for potential dental clinic expansion sites, pinpointing areas with strong market opportunity and limited existing coverage.

**Study Area:** Java and major cities across Indonesia <br>
**Role:** GIS Analyst <br>
**Status:** Completed <br>

---

## Methods & Tools

**Data Sources**

- Demographic data
- Point of Interest (POI) data
- Income data
- Road network data
- Competitor (existing dental clinic) locations

**Processing Steps**

1. Compiled demographic, income, POI, road, and competitor datasets covering Java and major Indonesian cities.
2. Built a hexagonal grid over the study area to standardize spatial analysis units.
3. Aggregated and scored each hexagon based on population density, income levels, accessibility (roads), and competitor proximity.
4. Produced a suitability map ranking hexagons by expansion potential for new dental clinic locations.

**Tools Used**

| Tool | Purpose |
|------|---------|
| ArcGIS Pro, QGIS | Hexagon grid generation and spatial scoring |
| ArcGIS Pro, QGIS | Data processing and mapping |
| OpenStreetMap | Data Source |

---
## Key Findings

- Identified high-suitability hexagon zones across Java and major Indonesian cities for dental clinic expansion.

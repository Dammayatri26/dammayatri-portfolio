## Overview

Built an interactive dashboard to track landscape restoration activities, spatial coverage, and key monitoring indicators, giving the RMEL team and stakeholders real-time visibility into project progress.

**Study Area:** Africa<br>
**Duration:** Oct 2025 – Jul 2026<br>
**Role:** Monitoring and Data Program Officer, RMEL Team<br>
**Status:** Completed

---

## Methods & Tools

**Data Sources**

- Field-collected monitoring data (via ArcGIS Field Maps)
- Centralized geospatial project database

**Processing Steps**

1. Integrated field-collected data from ArcGIS Field Maps into a centralized geospatial database.
2. Validated and cleaned spatial and attribute data to ensure reliable inputs for reporting.
3. Designed web maps in ArcGIS Online to visualize restoration activities and spatial coverage.
4. Built an interactive dashboard in ArcGIS Dashboard to track key monitoring indicators in real time.

**Tools Used**

| Tool | Purpose |
|------|---------|
| ArcGIS Field Maps | Field data collection |
| ArcGIS Online | Web map creation and data hosting |
| ArcGIS Pro | Cleaning and Spatial Data Processing |
| ArcGIS Dashboard | Interactive dashboard for monitoring indicators |

---
## Key Findings

- Enabled real-time tracking of restoration activities and spatial coverage for the team and stakeholder.
- Improved data accuracy and consistency by standardizing field data collection workflows.
